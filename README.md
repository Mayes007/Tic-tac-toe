# Tic Tac Toe - Smart AI Edition

A desktop-based Tic Tac Toe game built with Python and Tkinter. This application features multiple difficulty levels, including an unbeatable "Hard" mode powered by the Minimax algorithm.

## Instructions for Build and Use

Steps to build and/or run the software:

1.  **Environment:** Ensure Python 3.x is installed on your system.
2.  **Run:** Save the provided code as `tictactoe.py`.
3.  **Launch:** Open a terminal and run `python tic_tac_toe.py`.

Instructions for using the software:

1.  **Select Mode:** Use the buttons to toggle between "Player vs AI" or "Player vs Player".
2.  **Difficulty:** Choose "Easy", "Medium", or "Hard" to adjust the AI's logic.
3.  **Theme:** Click "Toggle Dark/Light Mode" to change the UI appearance.
